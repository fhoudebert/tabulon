Dans toutes les variantes du jeu de dames, les règles suivantes s'appliquent :

  • Les joueurs jouent à tour de rôle en déplaçant une pièce.
  • Les pièces peuvent se déplacer vers une case adjacente libre.
  • Une pièce adverse adjacente peut être capturée en sautant par-dessus (ce qui implique que la case située immédiatement derrière elle soit libre).

  • Lorsqu'il est possible d'effectuer plusieurs prises successives, elles sont réalisées au cours du même tour avec la même pièce.

Aux dames turques, les règles supplémentaires suivantes s'appliquent :

  • Le plateau et la position initiale sont les suivants :

  • Les pions se déplacent d'une case vers l'avant, horizontalement ou latéralement.
  • Les pièces promues, appelées dames, peuvent se déplacer d'un nombre quelconque de cases libres, horizontalement ou verticalement.
  • Après une capture, une dame doit s'arrêter immédiatement derrière la pièce capturée.
  • Les Blancs jouent en premier.
  • La prise est obligatoire.
  • Les pièces capturées sont retirées du plateau immédiatement après leur capture.
  • Si plusieurs séquences de prises sont possibles, le joueur doit choisir celle qui capture le plus grand nombre de pièces adverses.
  • Si une pièce atteint la dernière rangée au cours d'une séquence de prises, elle n'est promue que si cette séquence se termine sur cette dernière rangée.
  • Un joueur qui ne peut effectuer aucun coup perd la partie.
