Dans toutes les variantes de dames, les règles suivantes s'appliquent :

  • Les joueurs jouent à tour de rôle en déplaçant une pièce
  • Les pièces peuvent se déplacer vers une case vide adjacente
  • Une pièce adverse adjacente peut être capturée en sautant par-dessus celle-ci (ce qui implique que la case suivante soit libre)

  • Lorsque cela est possible, plusieurs captures doivent être effectuées durant le même tour (avec la même pièce)

Le gagnant est le premier joueur à ne plus avoir de coup légal possible : c'est-à-dire celui dont toutes les pièces ont été capturées ou sont bloquées.
