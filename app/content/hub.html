Tabulon is a multi-platform desktop application to play board games, based on the Jocly library

Jocly and Tabulon are released under AGPL v3.0 license.

Main features :

  • 2D and 3D user interface available
  • Fully resizeable game windows
  • Possible moves preview
  • Game import/export
  • Clocked games
  • Rules available for every game
  • Multi-windows: any number of games can be launched simultaneously
